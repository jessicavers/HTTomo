Parameter Sweeps templates#

Those templates demonstrate how to perform sweeps across multiple values a single parameter. See more on Parameter Tuning.

Manual variation of the Centre of Rotation in a range using !SweepRange functionality inside the reconstruction module.

Download

- httomo.data.hdf.loaders:
    standard_tomo:
      name: tomo
      data_path: /entry1/tomo_entry/data/data
      image_key_path: /entry1/tomo_entry/instrument/detector/image_key
      dimension: 1
      pad: 0
- tomopy.recon.algorithm:
    recon:
      data_in: tomo
      data_out: tomo
      center: !SweepRange
        start: 10
        stop: 50
        step: 10
      sinogram_order: false
      algorithm: gridrec
      init_recon: null
Manual variation of several values using Sweep functionality inside a filter.
